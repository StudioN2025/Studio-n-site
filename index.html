💬 P2P Мессенджер — РЕЛИЗ 0.1

💬
Мессенджер Studio N 🔥 v0.1 вышел!
Полностью децентрализованный P2P-мессенджер с шифрованием. Локальная версия — вы запускаете сервер на своём ПК и подключаетесь к друзьям напрямую.
🔒 E2EE P2P 🐍 Python ✅ релиз 25.03.2026
📡 Как работает: Вы скачиваете, запускаете сервер на своём ПК, даёте друзьям ваш IP — они подключаются. Вся история и файлы хранятся только у вас. 🔧 Сервер открываете вы, управление полностью ваше.
⬇️ Скачать релиз v0.1 (GitHub)
*Для работы нужен Python 3.8+, инструкция внутри архива
